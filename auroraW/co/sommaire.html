Insérer un sommaire automatique

Introduction

Le sommaire et la table des matières sont des listes organisées de titres pour une information rapide sur les chapitres d'un livre ou document, permettant de diriger rapidement le lecteur dans l'ouvrage à partir des informations principales.

Objectifs

  • Insérer un sommaire automatisé à partir de la hiérarchie des titres utilisés dans le document

Insérer un sommaire automatique

Consignes _ Insérer un sommaire automatisé

  • Reprendre votre document : "Méthodologie document complexe"

  • Créer une page vierge après la page de garde : Insertion=>Page=>Page vierge puis placer le curseur au début de cette nouvelle page

  • Insérer le sommaire automatique : Références=>Table des matières=>Table des matières personnalisé...=>Options

  • Sur cet écran d'options : attribuer le bon style au niveau hiérarchique souhaité

  • Mettre à jour le sommaire quand un changement est intervenu dans votre document : Références=>Mettre à jour la table

Une fois le sommaire automatique insérés, votre document doit ressembler à l'exemple ci-dessous :

Méthodologie document complexe SOMMAIRE.pdf

Une fois l'exercice fini...

  • Créer une copie de votre fichier en utilisant la fonction "Enregistrer sous" présent dans l'onglet "Fichier" de Word.

  • Nommer cette copie : "Méthodologie document complexe SOMMAIRE"

  • Déposer cette copie dans votre dossier partagé Google Drive, avec vos autres fichiers
