Gérer l'en-tête et le pied de page

Introduction

Définition :

Les en-têtes et les pieds de page sont affichés dans des zones spécifiques situées respectivement dans la marge supérieure et la marge inférieure d'une page ; ils peuvent comporter du texte ou des images. Les en-têtes et les pieds de page sont ajoutés au style de page actif, c'est-à-dire qu'ils sont insérés dans toutes les pages formatées avec le même style. En outre, vous pouvez ajouter différents champs, tels que des numéros de page et des titres de chapitre, dans les en-têtes et les pieds de page d'un document texte.

Objectifs

  • Insérer une entête pour la page de garde différente du reste du document

  • Insérer une numérotation de page automatique dans le pied de page

Consignes

  • Reprendre votre document : "Méthodologie document complexe"

Les éléments à insérer :

  • Dans l'en-tête de la page de garde : un logo de votre choix

  • Dans l'en-tête du document : votre Nom et Prénom ainsi que votre promotion

  • Dans le pied de page : le numéros de page

Une fois l'entête et le pied de page insérés, votre document doit ressembler à l'exemple ci-dessous :

Une fois l'exercice fini...

  • Créer une copie de votre fichier en utilisant la fonction "Enregistrer sous" présent dans l'onglet "Fichier" de Word.

  • Nommer cette copie : "Méthodologie document complexe ENTETE ET PIED DE PAGE"

  • Déposer cette copie dans votre dossier partagé Google Drive, avec vos autres fichiers
